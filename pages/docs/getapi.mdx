# API

All of the get(API) Methods

For Non Self host User : Before using the getAPI, Read this [Info](docs/info#usage) before you can use it.

## /getVideo

To get video from the hanime website
>/getVideo?id=(HANIME SLUG or VIDEO ID)

## /getVideo/premium

To get 1080p video from the hanime website, Use your account session_token to get the video

<Callout type="warning" emoji="❌">
  This method is not available for free user. You must require to buy Hanime.tv <a href="https://hanime.tv/premium">Premium</a> from your account to use this method.
</Callout>

To get account session_token go to [getSession](/docs/authentication/#authlogingetsession)

>/getVideo/premium?id=(HANIME SLUG or VIDEO ID)  

Before Requesting, Apply Token Key to the headers of the request.
