# API

All of the get(API) Methods

For Non Self host User : Before using the getAPI, Read this [Info](docs/info#usage) before you can use it.

## /getVideo

To get video from the hanime website

> /getVideo?id=(HANIME SLUG or VIDEO ID)

To get 1080p, You must add token to the headers and To get your account session token go to [getSession](/docs/authentication/#authlogingetsession)

**Headers**
> Token : (Your Account Token)

```vid
{
  "stream": [{
    ALL Stream List
  }]
  "downloadURL": "https://hanime.tv/downloads/<>",
  "url": "https://hanime.tv//videos/hentai/(HANIME SLUG VIDEO)"
}
```

## /getVideo/player

Create an embedded iframe or player url in the api with [nsplayer](https://player.nscdn.ml)

<Callout type="warning" emoji="❌">
  New Hanime Video Will be Work on this player but will not be available in the
  older upload due to the CORS error. If you want it to work, you downloads any cors extension.
</Callout>

> /getVideo/player?id=(HANIME SLUG or VIDEO ID)

```vid
{
  "downloadURL":"https://hanime.tv/downloads/<>",
  "iframe_api":"iframe with player url",
  "url":"https://hanime.tv/videos/hentai/<>"
  "player_url": "player_url",
  "main_url":"https://player.nscdn.ml"
}

```

## getLanding

Get information about landing in Hanime home page.

### getLanding/recent

> /getLanding/recent

```recent
{
  page: 0,
  results: [
    All Recent List
  ]
}

```

### getLanding/newest

> /getLanding/newest

```newest
{
  page: 0,
  results: [
    All Newest List
  ]
}

```

### getLanding/trending

> /getLanding/trending

```trending
{
  page: 0,
  results: [
    All Trending List
  ]
}

```

The Default Results are Monthly but You can specify trending time from Daily to Yearly

> /getLanding/trending?time=(day, week, month, 3_month, 6_month, year)&p=0

And The Result should be the same as the previous result.
