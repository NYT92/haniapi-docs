# Information

## Usage

If you plan to used this for your application, yes, you can.  
It is free to use and you dont have to pay for it. But There is a rate limit for each routes.

**Daily Limits : Unlimited**

| **Routes**             | **Rate Limits**    | **Methods** |
| ---------------------- | ------------------ | ----------- |
| /auth/login            | 180 req per minute | POST        |
| /auth/login/summary    | 180 req per minute | POST        |
| /auth/getcoins         | 180 req per minute | POST        |
| ---------------------- | -----------------  | ----------- |
| /getVideo              | 180 req per minute | GET         |
| /getInfo               | 180 req per minute | GET         |
| /getComment            | 180 req per minute | GET         |
| /getLanding            | 180 req per minute | GET         |
| ---------------------- | -----------------  | ----------- |
| /search                | 120 req per minute | GET/POST    |
| ---------------------- | -----------------  | ----------- |
| /browse                | 180 req per minute | GET         |
| ---------------------- | -----------------  | ----------- |
| /user                  | 180 req per minute | GET         |
| ---------------------- | -----------------  | ----------- |
| /community_upload      | 180 req per minute | GET/POST    |


## HTTP Response

| **Status Code** | **Description**       |
| --------------- | --------------------- |
| 200             | Success               |
| 401             | Unauthorized          |
| 404             | Not Found             |
| 405             | Methods Not Allowed   |
| 429             | Too many request      |
| 500             | Internal Server Error |
| 503             | Server Offline        |
