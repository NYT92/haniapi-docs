# Browse (Beta)

Find All Tags and Brands with Browse.

## /browse

Get All Tags / Brands in one request

> /browse

````br
{
  "brands": [
    All Brands
  ],
  "tags": [
    All Tags with images
  ],
  "video": "null"
}
````

## /browse With Filtering

To get the filtered tags and brands with videos

> /browse/type/(tag or brands)/page

### Params Requests

**Type** : hentai-tags, brands  
**Tag/Brands** : All Tags in Hanime  
For more [tags](search#tags) and [brands](search#brands), click on the text.  
**Page** : 0 is the default and Check the API how many pages are available.
